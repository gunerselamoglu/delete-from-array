# delete-from-array

>You can delete any element from the array

## Install

```
    npm install delete-from-array --save
```

## Usage


```
const deleteFromArray = require('delete-from-array');
const newArray = deleteFromArray([8,4,'g','n','r'], ['r',8]);
console.log(newArray); //4,g,n

```
